👨🏻‍💻Sales Dashboard for Regional Performance Data Analyst Portfolio Project

This project represents a complete, industry standard, end-to-end data analytics workflow, designed to mirror the real responsibilities of professional analysts in modern business environments. The project encompasses all critical stages of data analysis, from data preparation and modeling to insight generation, visualization, and reporting.

📌 Project Overview
This project demonstrates a complete data analytics workflow, starting from dataset ingestion to insights presentation. It covers data preparation, exploratory analysis, SQL querying, dashboard creation, and final reporting. The goal is to showcase end‑to‑end analytical capabilities using modern tools.

The goal of this project is to simulate a corporate-grade end-to-end data analytics workflow, demonstrating the ability to translate raw data into strategic business intelligence by:

✅ Power Query – Data cleaning, loading and transformation: Clean and transform the raw dataset for analysis.

✅ Data Analysis (SQL): Simulate business transactions, and run queries to extract insights.

✅ Visualization & Insights (Power BI): Build an interactive dashboard that highlights key patterns and trends, enabling stakeholders to make data-driven decisions and use DAX for 
   ✅Total Revenue & Units Sold, 
   ✅Year-over-Year (YoY) Growth,
   ✅ Month-over-Month (MoM) Growth
   ✅ EV (BEV) Penetration Trends
   ✅ Premium Segment Market Share
   ✅ Regional Performance (Europe, China, USA, Rest of World)
   ✅ How to create Field Parameters

✅ Report and Presentation: Write a clear project report summarizing your key findings and business recommendations. Prepare a presentation that visually communicates insights and actionable recommendations to stakeholders.



🔎 Steps
- Data Loading
- Imported dataset into Power Query for initial inspection.
- Exploratory Data Analysis (EDA)
- Identified missing values, outliers, and distribution patterns.
- Generated summary statistics and visualizations.
- Data Cleaning
- Removed duplicates, handled nulls, standardized formats.
- Ensured consistency for downstream SQL queries.
- SQL Queries (PostgreSQL)
- Created tables and relationships.
- Ran queries for aggregations, joins, and trend analysis.
- Dashboard (Power BI)
- Built interactive visuals (KPIs, charts, filters).
- Highlighted key insights for decision‑making.
- Report & Presentation
- Compiled findings into a structured report.
- Designed a professional PPT using Gamma for stakeholder presentation.

📈 Dashboard
- Contains interactive charts, slicers, and KPIs.
- Focus areas: trends, performance metrics, and actionable insights.
- Designed for clarity and recruiter‑friendly storytelling.
